# SeeFactory - AI Powered Movie Studio

**SeeFactory** is a cutting-edge web application that empowers users to create cinematic videos and storyboards using advanced AI agencies. Whether you're a filmmaker, storyteller, or creative enthusiast, SeeFactory provides the tools to turn your ideas into visual reality.

## üöÄ Features

*   **AI Director & Screenwriter**: Collaborates with intelligent agents to draft scripts, develop characters, and plan scenes.
*   **Visual Storyboard Generation**: Automatically generates consistent and high-quality images for each scene in your script.
*   **Video Continuation Mode**: Upload an existing video clip, and SeeFactory will extract the last frame to seamlessly generate a continuation of the narrative.
*   **Multi-Mode Creativity**:
    *   **Storyboard**: Focus on frame-by-frame visual planning.
    *   **Video Continuation**: Extend your existing footage.
    *   **Freeform**: Explore loose ideas and themes.
